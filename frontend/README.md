# Finance Tracker Web Application

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) application for tracking personal finances with authentication and OTP verification.

## Features

- ✅ User Authentication (Signup, Signin, Logout)
- ✅ OTP Verification for Email
- ✅ Password Reset with OTP
- ✅ Add Income/Expense Transactions
- ✅ View Transactions with Filtering & Sorting
- ✅ Financial Summary (Income, Expense, Balance)
- ✅ User Profile Management
- ✅ Responsive Design (Mobile + Desktop)
- ✅ JWT Token-based Authentication

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- Nodemailer for OTP emails
- bcryptjs for password hashing

### Frontend
- React.js
- React Router for navigation
- Context API for state management
- Axios for API calls
- Tailwind CSS for styling

## Prerequisites

- Node.js (v14 or higher)
- MongoDB (local or Atlas)
- Gmail account for sending OTP emails
